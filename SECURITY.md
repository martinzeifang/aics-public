# Security Policy

## Reporting Vulnerabilities

Bitte melde Sicherheitslücken **vertraulich** per E-Mail an
martin.zeifang@gmail.com — NICHT als öffentliches GitHub Issue.

Wir bestätigen den Eingang innerhalb von 48 Stunden und liefern
eine Erstanalyse innerhalb von 7 Tagen.

Public Disclosure erfolgt erst nach koordinierter Behebung.
