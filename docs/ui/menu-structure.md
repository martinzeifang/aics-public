# Menüstruktur – AI Compliance Suite

## Datei
| Eintrag | Funktion |
|---------|----------|
| Backup… | Öffnet den Backup-/Wiederherstellungsdialog |
| --- | |
| Schließen | Beendet die Anwendung |

## Bearbeiten
| Eintrag | Funktion |
|---------|----------|
| Projektvorgaben… | Öffnet die Projektkonfiguration des aktiven Moduls |
| Prompts bearbeiten… | Öffnet den Prompt-Editor des aktiven Moduls |

## Tools
| Eintrag | Funktion |
|---------|----------|
| KI-Einstellungen… | Globale und modulspezifische KI-Parameter (Modell, Temperatur, Tokens) |
| Nachweise… | Nachweisbibliothek verwalten (Dokumente, Tags) |
| Module verwalten… | Reihenfolge und Sichtbarkeit der Module anpassen |
| Fenstergröße… | Standard-Fenstergrößen je Modul einstellen |
| --- | |
| Debug-Modus | Aktiviert/deaktiviert erweiterte Protokollierung (Toggle) |

## Module (dynamisch)
Jedes aktivierte Modul erhält ein eigenes Menü mit modulspezifischen Aktionen,
z. B. „Kunden" (Kunden anlegen/bearbeiten), „BASO", „ICT", „Gutachten", „Risikobewertung".

## Ansicht
| Eintrag | Funktion |
|---------|----------|
| Dark Mode | Schaltet zwischen hellem und dunklem Farbschema um (Toggle) |
| --- | |
| Risikomatrix… | Öffnet das Risikomatrix-Fenster |
| --- | |
| Datenbanken… ▸ | Untermenü: direkter Zugriff auf alle SQLite-Datenbanken |

### Untermenü „Datenbanken"
Kunden · Evidence · CRA-Readiness · DSGVO-Readiness · NIS2-Readiness ·
Risikobewertung · Gutachten · BASO · ICT · Compliance-DB

## Hilfe

|---------|----------|
